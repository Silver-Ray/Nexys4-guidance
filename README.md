## 添加板卡文件

将`board`文件夹中的`nexys4_ddr`文件夹复制到安装路径下的`board_files`文件夹中（若`boards`文件夹下无`board_files`文件夹，则先创建后再复制文件夹），如`E:\Xilinx\Vivado\2023.1\data\boards\board_files`。
